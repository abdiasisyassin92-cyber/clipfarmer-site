2. What ClipFarmer does

ClipFarmer is a locally-run automation tool that:

  • Downloads royalty-free audio and video assets
  • Renders short-form vertical video clips using ffmpeg
  • Publishes the clips to TikTok, YouTube Shorts, and Instagram Reels on a scheduled basis
  • Retrieves basic post-performance analytics (views, likes) for internal optimisation

This application is not a service offered to third parties. It is a personal tool used solely by its developer.

3. Data collected through TikTok API

ClipFarmer uses the TikTok Content Posting API. The following data is accessed:

  • Creator profile information (username, display name, avatar) — used only to confirm that the correct TikTok account is authorised before publishing
  • Video publication capability — used to upload and post video clips on behalf of the authorised account
  • Post status and performance metrics (views, likes, shares) — used locally to rank content hooks by performance

All data retrieved through the TikTok API is processed locally on the developer's machine. No data is transmitted to any external server controlled by ClipFarmer.

4. Data we do NOT collect

  • We do not collect data from any TikTok user other than the authorised creator account
  • We do not read, store, or process followers' data, comments, messages, or profile information
  • We do not run a database of user accounts
  • We do not use cookies or tracking technologies on this website
  • We do not share any data with advertisers or data brokers

5. How data is stored

Any data retrieved from TikTok's API is stored locally on the developer's Windows machine at C:\clipfarmer\ in local JSON and log files. This data is:

  • Not uploaded to any cloud service
  • Not shared with third parties
  • Retained only for as long as operationally necessary

6. Access tokens

TikTok OAuth access tokens and refresh tokens are stored locally in config.json on the developer's machine. These tokens are never logged, printed, or transmitted to any endpoint other than TikTok's official API (open.tiktokapis.com).

7. Third-party services

ClipFarmer interacts with the following third-party APIs:

  • TikTok — Content Posting API (TikTok Privacy Policy)
  • YouTube / Google — YouTube Data API v3 (Google Privacy Policy)
  • Meta / Instagram — Graph API (Meta Privacy Policy)
  • Gumroad — used for product sales; governed by Gumroad's Privacy Policy
